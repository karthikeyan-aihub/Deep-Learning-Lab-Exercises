# Deep-Learning-Lab-Exercises

Complete implementation of **7 Deep Learning lab experiments** with **clean and error-free code**.  
Includes implementations of **CNN, RNN, Transfer Learning, Data Augmentation, and Data Pipelines**.

The programs were mainly tested using **Visual Studio Code (VS Code)**.  
You can also run them using **Jupyter Notebook** if you prefer.

---

# Experiments Included

## 1. Data Augmentation Pipeline Using TensorFlow
Implementation of image augmentation techniques such as **flipping, rotation, and scaling** to improve model performance.

## 2. Data Pipelining
Building efficient **data input pipelines** for preprocessing and loading datasets during model training.

## 3. Study of Deep Neural Networks
Implementation and understanding of a basic **Deep Neural Network (DNN)** architecture.

## 4. Sentence Classification using 1D CNN
Text classification using a **1D Convolutional Neural Network** to classify sentences into categories.

## 5. Age Prediction using 2D CNN
Predicting a person's age from facial images using a **2D Convolutional Neural Network**.

## 6. Transfer Learning and Fine Tuning
Using **pretrained models** and fine-tuning them for improved performance on a new dataset.

## 7. Document Classification using RNN
Implementation of a **Recurrent Neural Network (RNN)** model for document classification tasks.
